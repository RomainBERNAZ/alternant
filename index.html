Motivation

Livré avec de nombreuses options, l'alternant travaillera pour vous dans toutes les conditions.
D'un naturel curieux, il saura se tenir informé des avancées du métier que vous aurez paramétré lors de sa réception.
Malgré sa formation initiale dans l'immobilier, l'alternant semble mieux fonctionner dans le domaine du développement informatique.

Apprentissage

L'alternant est paramétré pour développer des applications web et web mobile, grace à sa formation bac+2 au sein de l'ENI école informatique.
Il maitrise plusieurs langages de programmation et possède bien évidemment assez de stockage interne pour en assimiler d'autres.
Pour l'instant l'alternant est livré avec les options suivantes :
- JAVA (+Framework STRUTS2)
- HTML / CSS
- JavaScript etc.

Avenir

L'alternant est livré pour une durée minimale d'un an. Pendant cette période il devra s'absenter quelques jours afin de mettre à jour ses connaissances.
Il est tout a fait possible de prolonger l'alternant après cette première année. Si celui-ci se sent bien chez vous, il s'engagera avec un immense plaisir sur le long terme.
